# Dinosaur Jump Game
A Scratch game inspired by the Chrome offline dinosaur game. 
Control a dinosaur and jump over obstacles to avoid collisions. This project was submitted as part of CS50 Problem Set 0.
## How to Play
1. Click the green flag to start the game.  
2. Click anywhere on the stage to activate keyboard input.  
3. Press the space bar to make the dinosaur jump.  
4. Avoid obstacles to increase your score. The game ends if the dinosaur touches an obstacle.
## Features
- Simple and intuitive controls  
- Score tracking for each obstacle passed  
- Collision detection ends the game  
## Technology
- Built entirely in [Scratch](https://scratch.mit.edu/)  
- Uses Scratch events, loops, and variables for game logic
